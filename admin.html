HOW TO USE THIS PANEL

  1. Check GitHub Issues for new tool submissions.
  2. Use the Add New Tool form below to generate the JSON code.
  3. Click Edit tools.json to open the file on GitHub.
  4. Paste the generated JSON into the `tools` array and commit changes.
  5. Close the submission issue on GitHub.
